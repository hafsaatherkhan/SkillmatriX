# SkillmatriX 🎯

An AI-powered career development platform that helps users identify skill gaps, build professional resumes, and receive personalized career roadmaps.

Built as a semester project for BS Software Engineering at SSUET.

---

## My Contribution

> This was a team project. My role was **Authentication & Backend Developer**.

- Built the complete authentication system using Spring Boot with token-based authorization
- Developed OTP-based password recovery with expiry timers, retry limits, and abuse prevention
- Integrated email services for OTP verification and user onboarding flows
- Designed backend logic for secure user data management and system reliability

---

## Features

- 📄 AI-powered resume builder
- 🔍 Skill gap analysis based on user profile
- 🗺️ Personalized career roadmap generation
- 🤖 Gemini API chatbot for intelligent recommendations
- 🔐 Secure authentication with OTP-based password recovery
- 👤 User onboarding and profile management
